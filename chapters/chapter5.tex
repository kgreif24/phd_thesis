\section{Unfolding as an Inverse Problem}
\label{sec:unfolding-intro}

% ---------------------------------------------------------------
% Sub-topics to cover:
%
% 1. The measurement problem
%    - We measure detector-level (reco) quantities; we want
%      particle-level (truth) distributions.
%    - The detector smears, distorts, and has finite acceptance:
%        x_reco = R(x_truth) + noise
%    - Unfolding inverts this: infer p(x_truth) from observed
%      p(x_reco).  This is an ill-posed inverse problem.
%
% 2. Binned unfolding and the response matrix
%    - Discretize x_truth and x_reco into bins.
%    - Response matrix R_{ij} = P(reco in bin i | truth in bin j).
%    - Estimated from simulation; direct matrix inversion is
%      numerically unstable (amplifies statistical fluctuations).
%    - Regularization is required.
%
% 3. Iterative Bayesian Unfolding (IBU / D'Agostini)
%    - Apply Bayes' theorem iteratively:
%        P(truth_j | reco_i) updated each iteration.
%    - Stopping criterion = regularization (early stopping).
%    - Widely used in HEP (RooUnfold); well understood.
%    - Limitations:
%        a) Binned → loses information, requires choice of binning.
%        b) Curse of dimensionality: multi-D unfolding is very costly.
%        c) Observable-by-observable: must redo for each quantity.
%    - These limitations motivate the AI-based approaches below.

\section{AI-Based Unfolding: Unbinned and High-Dimensional}
\label{sec:ai-unfolding-motivation}

% ---------------------------------------------------------------
% Sub-topics to cover:
%
% 1. The two key advantages
%    a) Unbinned: operate on individual events, not histograms.
%       → No binning bias; full statistical power preserved.
%       → Can histogram in any observable post-unfolding.
%    b) High-dimensional: unfold the full phase space jointly.
%       → Correlations between observables are correctly handled.
%       → One network, many downstream observables.
%
% 2. Analytic bottlenecks in LHC analyses that this alleviates
%    - Choosing optimal binning is non-trivial and analysis-specific.
%    - Multi-dimensional binned unfolding has combinatorial bin scaling.
%    - Detector correction factors (unfolding weights) that are
%      per-event rather than per-bin enable more flexible analyses.
%
% 3. Overview of the AI unfolding landscape
%    - OmniFold (Andreassen et al. 2019): density ratio iteration.
%    - INN-based methods (Bellagente et al.): normalizing flows.
%    - Generative unfolding: train a generative model p(truth|reco).
%      Examples: cINN, diffusion-based (VLD — this section).
%    - Bayesian deep learning approaches.
